# HR Dashboard - Candidate Interview & Onboarding Platform

A simple, password-protected internal HR dashboard for managing candidate interviews and onboarding processes.

## Features

✅ **Candidate Management**
- Add and track candidates
- Store candidate details (email, phone, position)
- Upload and download resumes
- Update candidate status through the pipeline
- Add internal notes

✅ **Interview Management**
- Schedule interviews with specific dates and times
- Assign interviewers (team members)
- Record interview types (Phone, Technical, HR, Final)
- Rate candidates (1-5 star system)
- Store interview feedback
- Track interview history

✅ **Onboarding Tasks**
- Create onboarding checklists for new hires
- Assign tasks to team members
- Set due dates
- Track task completion status

✅ **Activity Tracking**
- Log all candidate actions
- Track status changes
- Complete audit trail

✅ **Team Access**
- Password-protected dashboard
- Share single password with team

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`
5. Login with password: `admin123`

## Deployment

Deploy to Railway, Heroku, or any Node.js hosting platform.

For Railway:
1. Go to railway.app
2. Connect your GitHub repository
3. Add environment variables: PASSWORD, NODE_ENV
4. Deploy!
